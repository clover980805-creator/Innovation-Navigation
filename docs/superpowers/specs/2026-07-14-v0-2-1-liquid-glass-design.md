# V0.2.1 Liquid Glass Visual Upgrade — Design

## Purpose

在不改变 Innovation Navigation 的信息架构、工具链接、工具名称或上线状态的前提下，将当前偏磨砂的卡片升级为更具折射、透光和流动色彩的 iOS 26 风格液态玻璃导航界面。

## Scope

- 强化六张工具卡片的液态玻璃材质：玻璃厚度、边缘高光、局部折射和克制的动态色彩。
- 移除工具图标现有的黑色外圈，将每个图标置于统一尺寸的透明液态载体中。
- 将工具标题、描述与状态改为卡片底部居中排版，并优化中英文的字重、字距和换行。
- 重新设计页面分享图与站点图标，使其沿用导航页的深色背景、彩色折射和无黑边图标语言。
- 桌面横屏保持三列两行；手机端固定三列两行，六个入口可在首屏区域连续扫读。
- 验证 320、375、414、768 宽度及 1440×900、1920×1080、2560×1440 横屏布局。

## Out of Scope

- 不改动工具 URL、文案语义、产品状态、时钟和键盘导航逻辑。
- 不引入第三方框架、运行时依赖或新的部署方式。
- 不修改 VPS 或线上文件；本轮仅在本地分支完成修改和验证。

## Design Direction

**Chosen direction: Strong Liquid Glass.**

深色科技背景保留为光线来源，而不是把每张卡片做成单一半透明矩形。每张卡片由四层组成：低饱和透明基底、随工具色变化的内部光晕、顶部与侧边的不对称折射高光、以及悬浮图标后方的局部折射盘。悬停时仅移动高光和图标的 `transform` / `opacity`，不使用会造成眩晕的连续动画；减少动态效果模式下保留静态高光和即时焦点反馈。

## Layout and Responsive Behavior

- **Desktop (≥1061px):** 三列两行，卡片高度适中；大图标居中，底部文字居中。
- **Tablet (768–1060px):** 保持三列，缩小间距和卡片内边距；不回退为两列，确保入口顺序稳定。
- **Mobile (320–767px):** 固定三列两行；卡片使用较紧凑的统一高度，图标尺寸与文字大小按 `clamp()` 缩放。320px 仍不溢出，较长标题允许在卡内最多两行。
- **Landscape:** 卡片网格保持完整，不让页头或页脚挤压首行入口；横屏手机使用更低的卡片高度。

## Accessibility and Interaction

- 核心文字与其背景达到 WCAG AA 对比度；装饰光效不能降低标题、状态和箭头的可读性。
- 键盘焦点以清晰的亮环表示，不只依赖颜色变化。
- 现有方向键导航读取实际网格列数，响应式调整后继续有效。
- `prefers-reduced-motion: reduce` 关闭浮动和折射位移动画，仅保留静态视觉层与即时交互反馈。

## Files

- Modify: `index.html` — 调整服务卡片的语义结构、图标容器与分享/站点图标引用。
- Modify: `css/style.css` — 液态玻璃 token、卡片与图标样式、桌面/手机/横屏响应式规则。
- Modify: `production/index.html` — 与本地根页面保持发布包一致。
- Modify: `production/css/style.css` — 与本地根样式保持发布包一致。
- Create: `img/share/innovation-navigation-share-v0-2-1.webp` 及其 `production/` 对应文件 — 仅在现有分享图无法满足新图标语言时创建。

## Acceptance Criteria

1. 工具图标不再存在厚黑色外圈；六个图标拥有统一玻璃载体尺寸。
2. 卡片呈现可辨识的透光、折射和工具色高光，不退化为通用磨砂玻璃。
3. 手机 320、375、414px 均为三列两行，无横向滚动、重叠或截断的入口文字。
4. 桌面横屏的三列两行无裁切，文字与图标居中，交互焦点清晰。
5. 本地根目录与 `production/` 发布包的 HTML/CSS/所用静态资源保持一致。
6. 仅在本地分支工作，不执行线上部署。
